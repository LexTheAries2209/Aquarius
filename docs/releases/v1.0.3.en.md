# Aquarius V1.0.3 Release Notes

Release date: 2026-07-08

Aquarius V1.0.3 expands metadata export coverage and keeps the `Version + Build` display-name rule. This build uses `Version = 1.0` and `Build = 3`, so the app display name is `Aquarius V1.0.3`.

## Changes

- Added Adobe Premiere Pro XML export using an `xmeml version="4"` structure.
- Added Final Cut Pro 7 XML export for legacy `xmeml version="4"` workflows.
- Added Final Cut Pro XML / FCPXML export using `fcpxml version="1.9"`.
- Added Avid Media Composer ALE export as tab-delimited ALE text.
- Expanded the Metadata Export menu to cover DaVinci Resolve CSV, Premiere Pro XML, Final Cut Pro 7 XML, Final Cut Pro XML, and Avid ALE.
- Added GitHub repository information in Project Settings for release notes, downloadable builds, and Issue / Pull Request feedback.
- Kept `Aquarius旧版软件` as a local-only archive folder that is no longer uploaded to GitHub.
- Set the app `Display Name` to `Aquarius V1.0.3` and rebuilt the universal macOS app for Apple Silicon and Intel Mac.

## Verification Status

DaVinci Resolve CSV is the currently verified primary workflow.

Except for DaVinci Resolve CSV, Premiere Pro XML, Final Cut Pro 7 XML, Final Cut Pro XML / FCPXML, and Avid ALE have only passed local structure/format smoke tests. They have not yet been verified by importing into the corresponding production apps. Please treat them as testable export options for now, and review their output before relying on them in a critical delivery workflow.

## Install

1. Download and unzip `Aquarius-v1.0.3-macOS.zip`.
2. Move `Aquarius V1.0.3.app` to `Applications` or your own tools folder.
3. On first launch, if macOS Gatekeeper shows an internet-download warning, right-click the app in Finder and choose Open, or allow it from System Settings > Privacy & Security.
