# Aquarius V1.0.3 发布说明

发布日期：2026-07-08

Aquarius V1.0.3 扩展了元数据导出范围，并继续使用 `Version + Build` 的显示命名规则。当前构建为 `Version = 1.0`、`Build = 3`，因此 App 显示名为 `Aquarius V1.0.3`。

## 本次更新

- 新增 Adobe Premiere Pro XML 导出，生成 `xmeml version="4"` 结构。
- 新增 Final Cut Pro 7 XML 导出，面向 legacy `xmeml version="4"` 工作流。
- 新增 Final Cut Pro XML / FCPXML 导出，生成 `fcpxml version="1.9"` 文件。
- 新增 Avid Media Composer ALE 导出，生成 tab 分隔的 ALE 文本。
- 扩展“导出元数据”菜单，支持 DaVinci Resolve CSV、Premiere Pro XML、Final Cut Pro 7 XML、Final Cut Pro XML 和 Avid ALE。
- 在项目设置中加入 GitHub 仓库说明，方便查看版本说明、下载发布包或提交 Issue / Pull Request。
- 继续保持 `Aquarius旧版软件` 作为本地旧版归档目录，不再上传到 GitHub。
- 将 App `Display Name` 更新为 `Aquarius V1.0.3`，并重新构建 universal macOS app，支持 Apple Silicon 和 Intel Mac。

## 验证状态

DaVinci Resolve CSV 是当前已验证的主工作流。

除 DaVinci Resolve CSV 外，Premiere Pro XML、Final Cut Pro 7 XML、Final Cut Pro XML / FCPXML 和 Avid ALE 目前只通过本地结构/格式烟雾测试，尚未在对应工作软件中完成实际导入验证。它们适合先用于测试和反馈，不建议在没有复核的情况下直接作为关键交付链路的唯一依据。

## 安装

1. 下载并解压 `Aquarius-v1.0.3-macOS.zip`。
2. 把 `Aquarius V1.0.3.app` 放到 `Applications` 或自己的工具目录。
3. 首次打开时，如果 macOS Gatekeeper 提示来自互联网下载的 App，请在 Finder 中右键点击 App 后选择“打开”，或在“系统设置 > 隐私与安全性”中允许打开。
