# Aquarius V1.0.2 Release Notes

Release date: 2026-07-07

Aquarius V1.0.2 is a version-naming revision and rebuild release. This release adopts the new naming rule: the app `Display Name` shows a three-part version label composed from Xcode `Version` and `Build`. The current build uses `Version = 1.0` and `Build = 2`, so the app display name is `Aquarius V1.0.2`.

## Changes

- Set Xcode `Version` to `1.0` for the current major version line.
- Set Xcode `Build` to `2` for the second build in this version line.
- Set the app `Display Name` to `Aquarius V1.0.2`.
- Rebuilt the universal macOS app for Apple Silicon and Intel Mac.
- Updated README and release notes with the new download name and version policy.

## Feature Scope

This release keeps the V1.0.1 feature scope: burned-in metadata OCR, ROI presets, DaVinci Resolve CSV export, copy-and-rename outputs, optional QuickTime TMCD writing, optional MediaInfo CLI source metadata, and manual source timecode-rate selection for 23.976 / 29.97 NDF / 29.97 DF workflows.

## Install

1. Download and unzip `Aquarius-v1.0.2-macOS.zip`.
2. Move `Aquarius V1.0.2.app` to `Applications` or your own tools folder.
3. On first launch, if macOS Gatekeeper shows an internet-download warning, right-click the app in Finder and choose Open, or allow it from System Settings > Privacy & Security.
