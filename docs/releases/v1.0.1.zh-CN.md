# Aquarius V1.0.1 发布说明

发布日期：2026-07-07

Aquarius V1.0.1 是面向 QTake、现场监看录制、dailies 和代理文件整理流程的首个正式发布版本。它可以从画面烧录文字中识别文件名、卷号和源时间码，并把这些结果整理成 DaVinci Resolve 可用的元数据，或写回 MOV / QuickTime 的 TMCD 时间码轨道。

## 核心功能

- 导入单个视频、多个视频、文件夹，或直接拖拽素材到窗口。
- 在视频预览画面上调整 OCR 区域，分别识别文件名、卷号和时间码。
- 保存、重命名、导入和导出 ROI JSON 预设，方便适配不同 QTake 叠字布局。
- 支持单条素材提取，也支持当前列表批量提取。
- 多点抽样 OCR，并用一致性判断区分可信、需复核和失败状态。
- 支持自动源时间码帧率判断，也支持手动选择 23.976、24、25、29.97 NDF、29.97 DF 和 30。
- 检测视频帧率与源 TC 帧率不一致、疑似漂移、高风险识别和当前 TMCD 不一致等情况。
- 可选读取 MediaInfo CLI 输出，显示更完整的容器、编码、帧率、时长和当前 timecode 信息。
- 支持手动覆盖文件名、起始 TC、卷号和机位号。
- 导出 DaVinci Resolve 元数据 CSV。
- 可选把识别到的起始时间码写入 QuickTime TMCD 轨道。
- 可选复制到新文件夹并按识别或手动输入的文件名改名。
- 可选在复制改名后自动生成配套 DaVinci CSV，使用改名后的真实文件名作为 Resolve 匹配键。

## 下载和安装

1. 从 GitHub Release 下载 `Aquarius-v1.0.1-macOS.zip`。
2. 解压后把 `Aquarius V1.0.1.app` 放到 `Applications` 或你自己的工具目录。
3. 首次打开时，如果 macOS Gatekeeper 提示来自互联网下载的 App，请在 Finder 中右键点击 App 后选择“打开”，或在“系统设置 > 隐私与安全性”中允许打开。

## 系统要求

- macOS 13.0 或更新版本。
- 发布包不需要 Xcode。
- 可选依赖：MediaInfo CLI。安装后 Aquarius 可以显示更完整的源文件元数据；未安装时 OCR、CSV 导出、复制改名和 TMCD 写入仍可使用。

## 已知限制

- 当前正式支持 MOV / QuickTime 文件。
- OCR 效果依赖画面叠字清晰度和 ROI 位置，不同布局通常需要单独的 ROI 预设。
- 自动源 TC 帧率推断只覆盖 24、25 和 30；23.976、29.97 NDF 和 29.97 DF 需要手动指定。
- 29.97 DF 支持手动解析和 CSV 导出；QuickTime TMCD 写回仍处于保护状态，暂不写入非整数或 drop-frame TMCD。
- 直接写入源文件只支持已有可写 TMCD 轨道的 MOV / QuickTime 文件。
