# Aquarius V1.0.2 发布说明

发布日期：2026-07-07

Aquarius V1.0.2 是一次版本命名规则修订和重新构建发布。本次发布采用新的版本命名原则：`Display Name` 显示由 `Version` 和 `Build` 组合出的三段式版本名；当前构建为 `Version = 1.0`、`Build = 2`，因此 App 显示名为 `Aquarius V1.0.2`。

## 本次更新

- 将 Xcode `Version` 设为 `1.0`，表示当前大版本号。
- 将 Xcode `Build` 设为 `2`，表示当前大版本下的第 2 次构建。
- 将 App `Display Name` 统一为 `Aquarius V1.0.2`。
- 重新构建 universal macOS app，支持 Apple Silicon 和 Intel Mac。
- 同步 README 与发布说明中的下载文件名和版本说明。

## 功能状态

本次发布延续 V1.0.1 的功能范围，包括画面 OCR 元数据恢复、ROI 预设、DaVinci Resolve CSV、复制改名、可选 QuickTime TMCD 写入、MediaInfo CLI 辅助元数据读取，以及 23.976 / 29.97 NDF / 29.97 DF 的手动源时间码帧率选择。

## 安装

1. 下载并解压 `Aquarius-v1.0.2-macOS.zip`。
2. 把 `Aquarius V1.0.2.app` 放到 `Applications` 或自己的工具目录。
3. 首次打开时，如果 macOS Gatekeeper 提示来自互联网下载的 App，请在 Finder 中右键点击 App 后选择“打开”，或在“系统设置 > 隐私与安全性”中允许打开。
